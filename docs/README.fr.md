## Qu'est-ce que A11y Ally ?

**A11y Ally** est un assistant d'accessibilité web open source qui aide les développeurs à détecter et corriger les problèmes d'accessibilité sur n'importe quel site web. Il combine la puissance d'[axe-core](https://github.com/dequelabs/axe-core) avec des explications propulsées par IA pour produire des rapports exploitables avec des snippets de correction prêts à l'emploi.

### Trois outils en un :

| Outil | Description |
|-------|-------------|
| **🔖 Bookmarklet** | Outil navigateur en un clic — glissez dans les favoris, analysez toute page |
| **⌨️ CLI** | Scanner en ligne de commande Node.js pour pipelines CI/CD |
| **🌐 Web UI** | Site statique GitHub Pages avec démo et documentation |

## Fonctionnalités

- ✅ **80+ règles** d'accessibilité via axe-core (WCAG 2.1 A/AA/AAA)
- 🤖 **Explications IA** — descriptions claires et exploitables
- ✂️ **Snippets de correction** — code prêt à copier-coller
- 📊 **Rapports professionnels** — formats HTML, JSON et Markdown
- 🌍 **7 langues** — Anglais, Espagnol, Français, Allemand, Portugais, Chinois, Japonais
- 🆓 **Gratuit et open source** — Licence MIT
