## O que é o A11y Ally?

**A11y Ally** é um assistente de acessibilidade web de código aberto que ajuda desenvolvedores a encontrar e corrigir problemas de acessibilidade em qualquer site. Combina o poder do [axe-core](https://github.com/dequelabs/axe-core) com explicações de IA para produzir relatórios acionáveis com snippets de correção prontos para uso.

### Três ferramentas em uma:

| Ferramenta | Descrição |
|------------|-----------|
| **🔖 Bookmarklet** | Ferramenta de um clique — arraste para favoritos, analise qualquer página |
| **⌨️ CLI** | Scanner Node.js para pipelines CI/CD |
| **🌐 Web UI** | Site estático GitHub Pages com demo e documentação |
