## Was ist A11y Ally?

**A11y Ally** ist ein Open-Source Web-Barrierefreiheits-Assistent, der Entwicklern hilft, Barrierefreiheitsprobleme auf jeder Website zu finden und zu beheben. Es kombiniert die Leistung von [axe-core](https://github.com/dequelabs/axe-core) mit KI-gestützten Erklärungen.

### Drei Werkzeuge in einem:

| Werkzeug | Beschreibung |
|----------|-------------|
| **🔖 Bookmarklet** | Ein-Klick-Browser-Tool — in Lesezeichenleiste ziehen |
| **⌨️ CLI** | Node.js Kommandozeilen-Scanner für CI/CD |
| **🌐 Web UI** | Statische GitHub Pages Seite mit Demo |

---

## Funktionen

- ✅ **80+ Barrierefreiheitsregeln** via axe-core (WCAG 2.1)
- 🤖 **KI-Erklärungen** — klare, umsetzbare Beschreibungen
- ✂️ **Fix-Snippets** — kopierbereiter Code
- 📊 **Professionelle Berichte** — HTML, JSON, Markdown
- 🌍 **7 Sprachen** unterstützt
- 🆓 **Kostenlos & Open Source** — MIT-Lizenz

---

## Schnellstart

```bash
cd cli && npm install
node src/index.js scan https://example.com
node src/index.js demo
```
