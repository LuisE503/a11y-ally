⌨️ CLI Tool

Install globally with npm:

$ npm install -g a11y-ally-cli

Usage:

$ a11y-ally scan https://example.com
$ a11y-ally scan https://example.com -f html -o report.html
$ a11y-ally demo
$ a11y-ally batch urls.txt --output-dir ./reports
